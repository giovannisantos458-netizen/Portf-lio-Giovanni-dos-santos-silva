📝 Descrição do Projeto
O Vision Guard é um sistema de inteligência artificial focado em Classificação Visual para ambientes corporativos e industriais. O objetivo principal do projeto é automatizar a identificação de perfis profissionais e, crucialmente, monitorar a conformidade de segurança através da análise de imagens em tempo real.

Desenvolvido como parte da disciplina de Inteligência Artificial (2024.1) , o sistema utiliza modelos de aprendizado de máquina para categorizar colaboradores entre Liderança Executiva e Apoio Administrativo. O motor de visão computacional analisa padrões de vestimenta e o uso de equipamentos para validar se a classificação está correta ou se apresenta falhas críticas de segurança.
