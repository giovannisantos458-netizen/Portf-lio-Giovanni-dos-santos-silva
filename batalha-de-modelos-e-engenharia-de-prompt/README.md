

# ⚔️ Prompt Battle & XML Engine - Otimização de Modelos de Linguagem

## 📝 Descrição do Projeto

Este projeto consiste em um sistema de avaliação e refinamento de modelos de linguagem de grande escala (LLMs) através de técnicas avançadas de **Engenharia de Prompt** e estruturação de dados em **XML**. O objetivo principal é comparar o desempenho de diferentes modelos em tarefas de lógica e criatividade, utilizando tags estruturais para maximizar a precisão e a aderência às instruções fornecidas.

Desenvolvido como parte da disciplina de **Inteligência Artificial (2024.1)**, o sistema utiliza uma metodologia de "Batalha de Modelos" para identificar qual arquitetura de IA apresenta melhores resultados em cenários específicos. Através do uso de delimitadores XML, o motor organiza as instruções de sistema, variáveis de contexto e exemplos (few-shot prompting), permitindo que a IA processe informações complexas com maior consistência e menos alucinações.

### ⚙️ Funcionalidades Técnicas

* **Estruturação XML**: Implementação de tags personalizadas para organizar o raciocínio da IA e separar claramente as instruções dos dados.
* **Benchmark de Modelos**: Comparação direta entre diferentes versões de IA para validar a eficiência na resolução de problemas lógicos.
* **Refinamento de Contexto**: Uso de técnicas de *System Instructions* para definir personas e comportamentos específicos durante a interação.
* **Otimização de Saída**: Configuração de formatos de resposta (como JSON ou texto puro) para facilitar a integração com outros sistemas digitais.


Como você está explorando a **Engenharia de Prompt**, uma ótima aplicação prática seria integrar os princípios de **UX e Usabilidade** na forma como a IA interage com o usuário. Garantir que as instruções de sistema forcem a IA a fornecer feedbacks claros e a reduzir a carga cognitiva do usuário final contará muitos pontos na avaliação da sua disciplina!
