📐 SVG Flow - Gerador de Vetores Inteligente
📝 Descrição do Projeto
O SVG Flow é um motor de geração e manipulação de gráficos vetoriais (SVG) que utiliza algoritmos de geometria computacional e parametrização dinâmica. O objetivo principal é automatizar a criação de elementos visuais escaláveis, permitindo a geração de ícones, padrões e ilustrações complexas através de variáveis de entrada e lógica algorítmica.

Desenvolvido como parte da disciplina de Inteligência Artificial (2024.1), o sistema processa estruturas de dados geométricas para identificar pontos de ancoragem, curvas de Bézier e propriedades de preenchimento. O motor utiliza técnicas de otimização para garantir que o código XML final seja leve, responsivo e totalmente compatível com os padrões modernos da web.
