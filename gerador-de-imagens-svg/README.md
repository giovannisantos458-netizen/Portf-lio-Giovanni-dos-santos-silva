# 🎨 Blobmaker — Gerador de Formas SVG com IA
 
## 📝 Descrição do Projeto
Este projeto consiste em uma aplicação web interativa para geração de formas orgânicas em SVG de maneira dinâmica e personalizada. O objetivo principal é permitir que o usuário crie elementos gráficos modernos utilizados em interfaces web e projetos de UI/UX através de controles visuais simples e interativos.
 
Desenvolvido utilizando o **Google AI Studio** com suporte de Inteligência Artificial, o projeto explora conceitos de geração procedural de SVG, interatividade visual e desenvolvimento de aplicações modernas com JavaScript e Node.js. A aplicação permite ajustar propriedades como complexidade, contraste e formato das formas geradas, além de possibilitar exportação rápida dos SVGs criados.

## 🚀 Tecnologias Utilizadas

## 📊 Resultados e Aprendizados
O projeto apresentou resultados positivos na criação de uma ferramenta gráfica moderna e interativa.
* **Geração Dinâmica:** Criação procedural de formas SVG em tempo real.
* **Interatividade Visual:** Implementação de sliders e controles personalizados.
* **Exportação SVG:** Possibilidade de copiar e baixar os SVGs gerados.
* **UI Moderna:** Desenvolvimento de interface minimalista, limpa e responsiva.
* **Integração com IA:** Utilização do Google AI Studio para acelerar o desenvolvimento da aplicação.
