

# 🔎 Diagnóstico de Usabilidade - Avaliação de Interface

## 📝 Descrição do Projeto

Este projeto consiste em um sistema de auditoria e diagnóstico de usabilidade para interfaces digitais. O objetivo principal é identificar barreiras de interação e falhas de acessibilidade em sistemas diversos, fornecendo recomendações técnicas para otimizar a experiência do usuário e garantir a conformidade com padrões de mercado.

Desenvolvido como parte da disciplina de **Inteligência Artificial (2024.1)**, o sistema utiliza princípios de análise heurística e normas de acessibilidade (como a WCAG) para processar o diagnóstico de elementos visuais e funcionais. Através de um roteiro estruturado, a ferramenta permite documentar problemas de interface e sugerir correções baseadas em boas práticas de design, facilitando a criação de ecossistemas digitais mais inclusivos e eficientes.

### ⚙️ Funcionalidades Técnicas

* **Análise Heurística:** Avaliação sistemática de interfaces com foco em visibilidade do status do sistema e controle do usuário.
* **Auditoria de Acessibilidade:** Verificação de conformidade com padrões internacionais para garantir o acesso a pessoas com diferentes necessidades.
* **Dossiê de Diagnóstico:** Estruturação de relatórios técnicos que detalham problemas, gravidade e possíveis soluções.
* **Mapeamento de Barreiras:** Identificação de pontos de fricção na jornada do usuário que impedem a conclusão de tarefas.
